Textbookmania

A website that allows students to sell and buy textbooks.

Download this project as a .zip file Download this project as a tar.gz file

Overview

A Play application that provides an interface for students to buy and sell textbooks.

Installation

Download the zip file, unzip, cd into the directory, invoke the Play console, and execute the run command to see the application at http://localhost:9000

Usage

Use the navbar to buy and sell text books or view student information.
